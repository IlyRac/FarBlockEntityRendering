<h1>Far Block Entity Rendering</h1>

<p><strong>Configure how far block entities render - from 64 to 512 blocks!</strong></p>

<img src="https://cdn.modrinth.com/data/cached_images/53cc515ae50747f707284caa9fe7a2e957a6cb39.gif" alt="Rendering Block Entity" style="max-width:100%;">

<hr>

<!-- What the mod does -->
<details>
  <summary>🎯 What Does This Mod Do?</summary>
  <p>
    Tired of chests, signs, and other block entities disappearing just 64 blocks away? 
  </p>
  <p>
    <strong>Far Block Entity Rendering</strong> lets you customize the render distance for all block entities, extending it up to 512 blocks (32 chunks)!
  </p>
</details>

<!-- Features -->
<details>
  <summary>✨ Features</summary>
  <ul>
    <li><strong>All Block Entities</strong>: Affects chests, signs, beds, banners, and more</li>
    <li><strong>Performance Friendly</strong>: Choose your preferred balance between visibility and performance</li>
    <li><strong>Configurable Distance</strong>: Set block entity render distance from 64 to 512 blocks (4-32 chunks)</li>
    <li><strong>Instant Application</strong>: Changes take effect immediately - no restart required!</li>
    <li><strong>Easy Configuration</strong>: Built-in ModMenu support for seamless in-game configuration</li>
  </ul>
</details>
